# joulukalenteri

##Tarkoitus

Vanhuksille Tangram-peli mielenvirkistykseksi. Auttaa hienomotoriikan ja käsi-silmäkoordinaation treenaamisessa.

Pelissä tarkoitus muodostaa kaikista annetuista paloista mallin muotoinen kuvio.
